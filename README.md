# Tiwlink - Real-Time Chat Application

Tiwlink is a real-time chat application built with Spring Boot and WebSocket (STOMP).  
It allows users to send and receive messages instantly with a modern chat UI.

## Features
- Real-time messaging using WebSocket
- Typing indicator
- Chat bubbles UI
- Dark mode interface
- Smooth scrolling
- Enter key to send messages

## Tech Stack
- Java
- Spring Boot
- WebSocket (STOMP)
- HTML
- CSS
- Bootstrap
- JavaScript
